**Automate The Boring Stuff or "ATBS" for short.**

This is an app that let's you run different commands to tinker with your PC

**Installing**

**Windows:**
Download the exe from the latest release (you need python and win11toast for it to work)


**Linux:**
Download the AppImage, then launch it with ./ATBS_For_Penguins-x86_64.AppImage from the directory you downloaded it to.
